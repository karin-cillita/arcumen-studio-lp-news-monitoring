# Visual Branding Guidelines

### Visual posture
- Institutional, calm, precise
- Feels like: a well-run advisory firm, not a startup
- Avoids: hype, gradients, playful illustrations, consumer SaaS vibes

### Colors

Primary
- Deep Navy — #1A2B4C (use for: Headers, primary backgrounds, logo, key UI elements)

Secondary
- Warm Stone — #A69F94 (use for: Secondary text, borders, supporting elements)

Accent
- Muted Gold — #B8A04A (use for: CTAs, highlights, emphasis, brand moments)

Backgrounds
- Soft Ivory — #F7F5F0 (use for: Page backgrounds, cards, containers)

Text: 
- Charcoal - #2D2D2D (use for: Body text, paragraphs)


## Typography

### Primary Typeface: Playfair Display
Use for: 
- Headlines and titles
- Pull quotes
- Hero statements
- Report covers
- Presentation titles

### Body Typeface: DM Sans
Use for:

- Body text
- Navigation and UI elements
- Captions and labels
- Subheadings
- Buttons and CTAs

### Layout & spacing

- Prefer narrow reading widths (max ~720px for text)
- Large vertical spacing between sections
- Content should breathe; never feel dense
- One primary column; avoid complex grids

### Visual references (directional)

- Institutional consulting decks
- High-end editorial websites
- Calm fintech / legal tech interfaces
